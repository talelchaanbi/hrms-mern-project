# HRMS – Human Resources Management System

This is a **MERN stack web application** for managing human resources in a company.

## Project Idea
The goal of this project is to build a **Human Resources Management System (HRMS)** that allows companies to manage:
- Employees profiles
- Attendance tracking
- Leave requests
- Job positions and departments
- Recruitment and candidate applications
- Basic performance tracking

This system solves the problem of **manual and inefficient HR management** by providing a centralized digital platform.

## Main Features
- User authentication (Admin / Employee)
- Employee management (CRUD)
- Leave request management
- Dashboard with basic statistics
- Responsive user interface

## Technologies
- MongoDB
- Express.js
- React.js
- Node.js

## Team & Methodology

This project is developed by a **team of 8 members** using **Scrum methodology**.

### Roles
- **Scrum Master:** Manages Scrum process and team coordination
- **Product Owner:** Defines product vision, backlog, and priorities
- **Frontend Team (2 members):** UI/UX, React components, integration
- **Backend Team (2 members):** REST API, database, business logic
- **Testing Team (2 members):** Test cases, manual testing, bug reporting

### Work Organization
- Development is organized in **sprints** with planning and review meetings
- Tasks are divided by teams (Frontend / Backend / Testing)
- GitHub repository is used for version control and issue tracking
